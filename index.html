Safari Extension
iPhone · iPad · Mac — one tap to save
Recommended
A One-time setup (2 min)
1
Open the Shortcuts app → tap + → name it "Save to Reader".
2
Tap Add Action → search "URL" → pick Get URLs from Input.
3
Tap + again → search "Get contents" → pick Get Contents of URL. Tap the URL field, set it to:

Tap Show more → Method: GET (default is fine).
4
Tap the shortcut name at the top → toggle Use as Quick Action ON → enable Show in Share Sheet.
5
Done! In Safari (or any app), tap the Share button → scroll down → tap Save to Reader.
Works in: Safari, Chrome, WhatsApp, Telegram, iMessage, Twitter/X, Reddit, Mail — any iOS app with a share button.
iPad setup is identical to iPhone above, plus you get a toolbar button:
1
Follow the iPhone steps above to create the Save to Reader shortcut.
2
In Safari on iPad, tap the Extensions icon in the address bar → your shortcut appears there for one-tap saving without opening the share sheet.
3
Or tap the Share button in the toolbar → Save to Reader.
Tip: On iPadOS you can pin the shortcut to the Safari toolbar for even faster access.

B Mac Share Menu (alternative)
1
Open System Settings → Privacy & Security → Extensions → Share Menu.
2
Enable Shortcuts here, then create a Mac shortcut (same steps as iPhone above). It then appears in Safari's Share menu.

Telegram Bot Any device

Create a private Telegram bot. Send it any URL from your phone, desktop, or tablet and it gets added to your Reader automatically via a free Pipedream webhook.

1
In Telegram, open @BotFather → send /newbot → follow the steps → copy your Bot Token.
2
Go to pipedream.com (free). Create a workflow → HTTP trigger → Node.js step. Copy the webhook URL.
3
In the Node.js step, paste this code:
export default defineComponent({ async run({ steps, $ }) { const msg = steps.trigger.event.body?.message?.text || ''; const match = msg.match(/https?:\/\/[^\s]+/); if (!match) return $.respond({ status: 200, body: 'no url' }); const readerUrl = 'PASTE_YOUR_READER_URL_HERE'; await fetch(readerUrl + '?add=' + encodeURIComponent(match[0])); return $.respond({ status: 200, body: 'saved' }); } });
4
Register the webhook: open this URL in your browser (replace both tokens):
https://api.telegram.org/botTOKEN/setWebhook?url=YOUR_PIPEDREAM_URL
Email Forwarding Any email client

Get a dedicated email address. Forward any email with a link and the URL auto-saves to your Reader.

1
Go to zapier.com (free) → New Zap → Trigger: Email by Zapier → gives you a @robot.zapier.com address.
2
Action: Webhooks → GET. URL: your Reader URL + ?add= + first URL from email body.
